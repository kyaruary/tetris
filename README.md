# 俄罗斯方块
- 配置项
- 单一职责

点（point） --> 小块（square） --> 俄罗斯块（tetris）
```
    方块的逻辑描述 不涉及具体视图展示
```
规则类
```
    提供游戏规则静态方法
    1. 边界判定方块可否移动
    2. 移动方块到指定地点 / 左右下旋转掉落
    3. 判断方块是否重叠
    4. 删除方块
```

游戏类
```
    控制游戏流程
    1. 游戏初始化
    2. 控制方块移动 / 自动下落
    3. 游戏状态管理
    4. 积分
    5. 消除方块
```

视图类
```
    将方块的逻辑描述转化为具体展示（dom/canvas）
```
